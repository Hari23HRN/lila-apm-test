# LILA BLACK — Player Journey Visualizer

**Live URL:** https://candid-gnome-b2ab14.netlify.app/

## Tech Stack
- Python (PyArrow, Pandas) for data processing
- Vanilla HTML/JS + Canvas API for the frontend
- Static hosting on Netlify

## Setup
No build step needed. The data is pre-processed.
To run locally: open index.html in a browser (requires a local server due to JSON loading).
To deploy: drag the folder to Netlify Drop.
